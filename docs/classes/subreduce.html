Class SubReduce

Clase de especificación de suma

La Clase AddReduce concreta el método computeAcc(), que es donde se calcula el valor del acumulador, mediante una resta entre el valor previo del acumulador y el valor numerico copiado actualmente

Esta toma por defecto en el acumulador el valor cero
